# LearnWord - 桌面端单词学习应用

一款基于 Python 和 PySide6 构建的桌面端单词学习应用，采用科学的多阶段学习法，助您高效记忆和巩固词汇。

## ✨ 核心功能概览

- 三阶段学习法： 在“学习”模式中，采用递进式三阶段学习法，确保单词从“词义认知”到“完整拼写”的全面掌握。

- 多模式切换： 提供学习 (Learn)、复习 (Review) 和测试 (Test) 三种模式，覆盖记忆周期的不同环节。

- 进度自动管理： 自动记录每个单词的当前学习阶段、熟练度及测试得分，并进行数据持久化。

- 灵活词库支持： 支持通过设置界面导入用户自定义的 CSV 格式单词库。

- 自动更新与初始化：

>- 主程序启动时和手动点击时，会自动检查远程版本更新。

>- 如果本地词库文件缺失，会自动尝试从网络下载默认词库。

>- 现代化界面： 采用 QSS 样式表进行美化，提供清晰舒适的黑色主题 UI。

## 📚 三阶段学习机制 (Learn Mode)

LearnWindow 实现了核心的三阶段学习逻辑，根据用户的回答结果动态调整单词的阶段。

### 阶段 1：词义认知

#### 目标: 确认对单词词义的基本认知。
>-交互方式: 词义选择题（四选一）。显示单词，选择正确的释义。
>- 答对处理: 进入下一阶段 (Stage + 1)。
>- 答错处理: 停留在本阶段，重新学习。

### 阶段 2：拼写/词义自测

#### 目标: 确认单词的初步掌握程度。
>- 交互方式: “认识”/“不认识”自测。显示单词和释义，用户自判断是否掌握。
>- 自测成功（点击“下一个”）: 进入下一阶段 (Stage + 1)。
>- 自测失败（点击“我记错了”）: 退回阶段 1。

### 阶段 3：拼写掌握

#### 目标: 掌握单词的完整拼写。

>- 交互方式: 拼写填空题。根据释义和带有下划线的提示，完成完整拼写。
>- 拼写正确: 标记为已学完 (learned = True)。
>- 拼写错误/我不会: 退回阶段 1。

## ⚙️ 运行环境与安装

### 依赖项
本应用基于 Python 3 运行，需要以下库：

- PySide6：用于构建 GUI 界面。

- requests：用于实现版本检查和默认词库网络下载。

-  建议在虚拟环境中安装
>- pip install PySide6 requests


### 运行程序

请确保所有程序文件（main.py, learn_window.py, vocab_model.py, setting_window.py 等）位于同一目录下，然后执行：

- python main.py


### 导入词库

- 启动程序后，点击主界面的 设置 按钮。

- 在设置窗口中，找到 导入新词库 选项。

- 选择您的 CSV 词库文件。

>- **词库格式要求： CSV 文件应包含 单词, 词性, 释义, 例句 等字段，以匹配程序内部的数据模型 (WordItem)。**

## 📂 代码文件结构

以下是主要文件的功能描述：

- main.py: 应用程序主入口。初始化应用，包含主窗口 (MainWindow) 和更新检查逻辑。

- vocab_model.py: 数据层。管理所有单词数据、学习进度、应用设置的加载、保存和状态更新。

- learn_window.py: 负责实现三阶段学习模式的界面和交互逻辑。

- review_window.py: 负责实现已学单词复习模式的界面和逻辑。

- test_window.py: 负责实现测试模式的界面和逻辑，包括记录测试得分。

- setting_window.py: 负责应用设置界面，处理词库导入、数量限制调整等。
